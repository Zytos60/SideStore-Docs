# 創建一個來源

SideStore 完全兼容 AltSources。官方的 AltStore [來源文檔](https://faq.altstore.io/developers/make-a-source) 提供了創建您自己的 AltSource 所需的所有詳細信息。

在編輯您的來源時，強烈建議使用 [AltSource Browser](https://altsource.by.lao.sb/browse) 創建並查看您的更改草稿，以便可視化顯示您的來源並進行檢查。

# 分發您的來源

您可以使用以下模板創建一個鏈接，來自動添加您的 AltSource：sidestore://source?url=[source url]

有關更多信息，請參閱 [URL Schema](/docs/advanced/url-schema)。通常建議以此格式創建您的 AltSource 鏈接，然後將其放入鏈接縮短器，例如 ```tinyurl.com```。然而，請謹慎選擇信任的鏈接縮短服務。
