# URL 模式

以下是使用 URL 模式與 SideStore 進行互動的清單。這些模式非常實用，因為它們允許開發人員只需放置一個一鍵連結，即可方便地使用。

目前，使用 URL 模式可以執行的操作僅限於安裝應用程式和來源，但未來將會擴展以接受多種輸入並執行更多樣化的任務。

## 安裝遠端託管的應用程式


```
sidestore://install?url=[download url]
```


## 添加 AltSource

```
sidestore://source?url=[source url]
```
